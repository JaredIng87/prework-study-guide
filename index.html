HTML

  • Element - the entire element from opening tag to closing tag
  • Tag - what appears inside the angle brackets.
  • !Doctype - document type declaration
  • html - represents the start of the HTML document
  • lang - language
  • head - contains information about the webpage
  • title - represents document title
  • style - CSS styles
  • base - base URL
  • link - external resource
  • script - client-side script
  • meta - metadata/ character set, description, keywords, author, viewport
  • body - represents the visible content shown to the user
  • header - appears at the top of the page and usually contains a navigation menu and/or introductory content
  • footer - appears at the bottom of the page and usually contains the author, copyright, contact, sitemap, and navigation
  • h1-h6 - heading level
  • section - create new section. Must have class and id
  • p - paragraph
  • ul - unordered list
  • ol - ordered list
  • li - list item
  • img - image
  • br - break/empty space
  • a - anchor/create link
  • div - create a division in the page
  • span - create an inline division

CSS

  • margin - amount of space around the outside of an element
  • padding - amount of space around the content of an element
  • * - wildcard selector
  • height - sets element height
  • width - sets elements width
  • background-color - sets elemnts background color
  • color - sets elements color
  • text-align - sets placement of element text
  • display - sets placement of image
  • font-size - sets font size

Git

  • git status - checks what branch we are currently on
  • git checkout - switches working branch
  • git checkout -b - creates new branch
  • git add -A - add all changes
  • git commit -m "message" - commit changes to repo
  • git pull origin main - syncs local branch to remote branch "Github"
  • git push origin - pushes changes to remote branch

JavaScript

  • Control flow - the order in which a computer executes code in a script.
  • var - variable: a named container that allows us to store data in our code.
  • console.log - output a message to console
